# HerCure

A comprehensive women's health app powered by AI for personalized health tracking and guidance.

## Features

- **User Authentication**: Secure login and signup with JWT tokens.
- **Health Assessments**: Comprehensive evaluations for hormonal, reproductive, heart, and menstrual health.
- **AI Health Assistant**: Chat with an AI-powered assistant for personalized insights and answers.
- **Symptom Logging**: Daily logging of symptoms, mood, and health metrics.
- **Health Dashboard**: Visual dashboard with health scores and overall status tracking.
- **Notifications**: Automated reminders and health alerts.
- **Profile Management**: User profile with health history and preferences.

## Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **Backend**: Node.js, Express.js, MongoDB
- **AI Integration**: Google Gemini API
- **Authentication**: JWT, bcrypt
- **Other**: Axios, Lucide React, React Speech Recognition

## Setup

### Prerequisites
- Node.js (v16+)
- MongoDB
- npm or yarn

### Backend Setup
```bash
cd backend
npm install
# Set up environment variables in .env (see .env.example if available)
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## Usage

1. Sign up or log in to your account.
2. Complete initial health assessment.
3. Use the dashboard to view health scores and quick actions.
4. Log daily symptoms and interact with the AI assistant.
5. Receive notifications and track progress over time.

**Note**: This app provides AI guidance only and is not a substitute for professional medical advice.
